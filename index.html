Preface

SuperCandy Studio ("we") is committed to protecting user privacy while providing high-quality services. This Privacy Agreement outlines how we handle personal information in compliance with applicable laws including the GDPR and Chinese regulations.

Key Definitions

  • We/Our: SuperCandy Studio, including our affiliates and partners
  • You/User: Individual using our mobile applications and services
  • Game Data: Includes character data, virtual items, behavioral logs, and transaction records

1. Information We Collect

Data Collection Methods

  • Explicitly provided information (registration, payment)
  • Automatically collected data (device info, IP address)
  • Third-party service data (analytics providers)

2. Data Usage & Sharing

Permitted Uses

  • Service delivery and improvement
  • Security monitoring and fraud prevention
  • Marketing communications (with consent)

3. Data Storage Practices

Security Measures

  • TLS 1.2+ encryption for data in transit
  • AES-256 encryption for data at rest
  • Regular third-party security audits

4. Security Protocol

Incident Response

  • 72-hour breach notification to authorities
  • User notification via in-app alerts
  • Forensic analysis and system hardening

Cookies & Tracking Technologies

Cookie Usage

  • Session management (PHPSESSID)
  • Analytics (Google Analytics)
  • Personalized ads (AdMob)
